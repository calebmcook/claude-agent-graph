    Topology Utilities¶

    Graph topology detection and validation.

    Topology detection and validation utilities.

    This module provides functions for detecting and validating graph topologies using NetworkX algorithms. It supports detection of trees, DAGs, chains, stars, and cycle graphs.

    Check if a graph is a linear chain.

    Parameters:¶

    claude_agent_graph.topology.is_star(graph: DiGraph) → bool[source]¶

    Check if a graph is a star topology.

    A star has one central node (hub) with high fanout or fanin (>1), connected to all other nodes (spokes), with no connections between spokes.
