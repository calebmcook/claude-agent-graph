    Agent Session Management¶

    Agent lifecycle and session management.

    Agent session management for claude-agent-graph.

    This module manages ClaudeSDKClient instances for agent nodes, handling lifecycle (start, stop, restart) and error recovery with retry logic.

    class claude_agent_graph.agent_manager.AgentSessionManager(graph: AgentGraph)[source]¶

    Bases: object

    Manages ClaudeSDKClient instances for agent nodes.

    Handles agent lifecycle (creation, start, stop, restart) and provides error recovery with exponential backoff retry logic.

    RETRY_DELAYS = [1.0, 2.0, 4.0]¶
